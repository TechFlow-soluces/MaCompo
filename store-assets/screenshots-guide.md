# Guide des Screenshots pour les Stores

## Google Play Store

### Exigences techniques
- **Nombre** : Minimum 2, maximum 8 screenshots
- **Format** : PNG ou JPEG (recommandé: PNG)
- **Résolution** :
  - Smartphone : 1080 x 1920 pixels (16:9)
  - Tablette 7" : 1200 x 1920 pixels
  - Tablette 10" : 1600 x 2560 pixels

### Screenshots recommandés (par ordre)

1. **Dashboard - Vue d'ensemble**
   - Titre : "Gérez toutes vos tactiques"
   - Montrer le dashboard avec plusieurs tactiques créées

2. **Terrain tactique - Composition**
   - Titre : "Créez vos formations facilement"
   - Montrer le terrain avec des joueurs positionnés en formation 4-3-3

3. **Édition de joueur**
   - Titre : "Personnalisez chaque joueur"
   - Montrer la modal d'édition d'un joueur avec ses informations

4. **Génération de convocation**
   - Titre : "Générez vos feuilles de match"
   - Montrer la liste de convocation générée

5. **Création de nouvelle tactique**
   - Titre : "Organisez vos compositions"
   - Montrer la modal de création de tactique

## App Store (iOS)

### Exigences techniques
- **Nombre** : Minimum 1, maximum 10 screenshots
- **Format** : PNG ou JPEG
- **Résolution iPhone** :
  - iPhone 6.7" : 1290 x 2796 pixels
  - iPhone 6.5" : 1242 x 2688 pixels
  - iPhone 5.5" : 1242 x 2208 pixels
- **Résolution iPad** :
  - iPad Pro (12.9") : 2048 x 2732 pixels
  - iPad Pro (11") : 1668 x 2388 pixels

### Screenshots recommandés
Même ordre que Google Play Store

## Conseils pour capturer les screenshots

1. **Utiliser un appareil/émulateur réel**
   - Android : Émulateur Android Studio avec Pixel 5 (1080x1920)
   - iOS : Simulateur Xcode avec iPhone 14 Pro Max

2. **Préparer des données de démo**
   - Créer 3-4 tactiques avec des noms parlants (4-3-3 Offensive, 4-4-2 Défensive, etc.)
   - Remplir les formations avec des noms de joueurs réalistes
   - Utiliser des numéros de maillot variés (1-11)

3. **Capturer les screenshots**
   - Android : Power + Volume Down
   - iOS : Side Button + Volume Up

4. **Ajouter des annotations (optionnel)**
   - Utiliser un outil comme Figma ou Canva
   - Ajouter des titres explicatifs en haut
   - Ajouter des flèches ou highlights si nécessaire

## Feature Graphic (Google Play uniquement)

### Exigences
- **Dimensions** : 1024 x 500 pixels
- **Format** : PNG ou JPEG
- **Poids** : Maximum 1 MB

### Contenu suggéré
- Logo de l'app à gauche
- Titre "MaCompo" au centre
- Slogan : "Vos tactiques de football, simplement"
- Image de terrain de football en arrière-plan (avec transparence)
- Couleurs : Vert football (#2ecc71) et noir

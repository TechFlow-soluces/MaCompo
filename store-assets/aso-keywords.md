# Stratégie ASO (App Store Optimization) - MaCompo

## Qu'est-ce que l'ASO ?
L'App Store Optimization est l'équivalent du SEO pour les applications mobiles. C'est l'optimisation de votre fiche app pour apparaître dans les recherches et améliorer le taux de téléchargement.

## Configuration par plateforme

### App Store (iOS)
**Limite : 100 caractères, séparés par virgules (sans espace)**

**Option 1 (Français focus)** :
```
football,tactique,compo,formation,coach,terrain,équipe,entraîneur,foot,sport,tableau,stratégie
```

**Option 2 (Mix FR/EN)** :
```
football,tactique,compo,formation,coach,terrain,équipe,tactics,lineup,soccer,sport,strategy
```

**Recommandation** : Option 1 si vous ciblez principalement la France
Option 2 si vous visez aussi les pays anglophones

### Google Play (Android)
Google Play n'a pas de champ "mots-clés" dédié. L'optimisation se fait via :
1. **Le titre** (30 caractères max après le nom de l'app)
2. **La description courte** (80 caractères)
3. **La description longue** (4000 caractères max)

**Titre optimisé** :
```
MaCompo - Tactiques Football
```

**Description courte optimisée** :
```
Créez vos tactiques de football facilement - Tableau tactique interactif
```

**Mots-clés à intégrer naturellement dans la description longue** :
- Formation football
- Coach football
- Entraîneur
- Composition d'équipe
- Tableau tactique
- Terrain de football
- Stratégie football
- Convocation
- Feuille de match
- Schéma tactique
- Plan de jeu

## Facteurs de classement ASO

### 1. Facteurs On-Metadata (sous votre contrôle)
- [x] **Nom de l'app** : MaCompo
- [x] **Sous-titre/Description courte** : Optimisé avec mots-clés
- [x] **Mots-clés** (iOS) : Liste préparée
- [x] **Description longue** : Contient naturellement les mots-clés principaux
- [x] **Catégorie** : Sports + Productivité
- [ ] **Screenshots** : À préparer avec texte descriptif
- [x] **Icône** : Générée et optimisée

### 2. Facteurs Off-Metadata (pas directement contrôlables)
- **Nombre de téléchargements** : Favorise le classement
- **Évaluations et notes** : Très important (viser 4.5+)
- **Taux de rétention** : % d'utilisateurs qui gardent l'app
- **Engagement** : Temps passé dans l'app
- **Taux de désinstallation** : À minimiser

### 3. Conseils pour améliorer ces facteurs
1. **Qualité de l'app** : Interface fluide, sans bugs
2. **Demander des avis** : Après 3-5 utilisations réussies
3. **Mises à jour régulières** : Montrer que l'app est maintenue
4. **Support utilisateur** : Répondre aux commentaires
5. **Partage social** : Faciliter le partage de tactiques (future feature)
